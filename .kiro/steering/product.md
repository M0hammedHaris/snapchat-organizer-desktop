# Product Overview

Snapchat Organizer Desktop is a professional desktop application for downloading and organizing Snapchat memories locally. The application provides a comprehensive solution for users who want to manage their Snapchat data exports with advanced features like overlay compositing, GPS metadata preservation, and timezone conversion.

## Core Features

- **Download memories** directly from Snapchat HTML exports with configurable options
- **Organize chat media** by contact and date using intelligent 3-tier matching
- **Remove duplicates** using SHA256 hashing with 99%+ accuracy
- **Verify file integrity** to detect corrupted or damaged media
- **Organize by year** using EXIF metadata and file timestamps
- **Fix timestamps** by syncing EXIF data to file modification times
- **Comprehensive help system** with step-by-step Snapchat data download instructions

## Business Model

Freemium + Subscription model with three tiers:
- **Free Tier:** Organize up to 100 files/month
- **Pro ($9.99/mo):** Unlimited files + overlay compositing + GPS embedding
- **Premium ($19.99/mo):** All Pro features + advanced analytics + cloud backup

## Target Audience

Snapchat users ages 16-35 with large data exports who want to preserve and organize their memories locally with professional-grade tools.
